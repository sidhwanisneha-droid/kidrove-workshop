## 📌 Project Overview

This project is a landing page for the AI & Robotics Summer Workshop designed for children aged 8–14 years.

The application provides workshop information, learning outcomes, FAQs, and a registration form connected to a backend API.

## ✨ Features

### Frontend

* Responsive Hero Section
* Workshop Details Section
* Learning Outcomes
* 4-Week Learning Journey
* FAQ Accordion
* Registration Form
* Loading States
* Form Validation
* Modern UI using Tailwind CSS

### Backend

* Express.js API
* POST /api/enquiry
* Required Field Validation
* Success Response Handling

## 🛠️ Tech Stack

### Frontend

* React.js
* Vite
* Tailwind CSS

### Backend

* Node.js
* Express.js
* CORS
